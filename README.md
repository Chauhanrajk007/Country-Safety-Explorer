# Country-Safety-Explorer

A clean, user-friendly web application built with a focus on UI/UX and system clarity.

## Features
- Responsive design (mobile + desktop)
- Simple, intuitive user flow
- Frontend-first architecture
- Backend-ready structure

## Tech Stack
- HTML
- CSS
- JavaScript
- (Planned) Node.js + MongoDB

## How to Run
Open `frontend/index.html` in browser.

## Design Philosophy
Frontend shows intent, backend enforces logic, database stores truth.

## Future Improvements
- Backend integration
- Authentication
- Data persistence
